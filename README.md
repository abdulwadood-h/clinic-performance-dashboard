# Clinic Performance Dashboard

This project is an interactive Power BI dashboard built to analyze clinic performance and patient metrics. It provides quick insights into patient trends, department activity, and overall clinic statistics.

## Key Metrics
- Patient Count
- Bed Nights
- Average Age
- Daily Average Patients
- IP Percentage

## Visualizations
- KPI cards for overall metrics
- Bar chart: Daily average patients by department
- Donut chart: Patient count by gender

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Purpose
The goal of this project is to demonstrate data analysis and dashboard design skills using Power BI.
